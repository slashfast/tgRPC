# tgRPC

## Getting started

Clone repo:

```bash
git clone https://github.com/slashfast/tgRPC.git
cd tgRPC
```

Fill in all empty environment variables in the `.env` file:

```bash
BOT_TOKEN=
RPC_ENDPOINT=127.0.0.1
RPC_PORT=7000
RPC_USER=
RPC_PASSWORD=
WALLET_PASSWORD=
API=https://mempool.space
USER_ID=
DATA_PATH=data.json
```

And almost done:

```bash
python3 main.py
```
